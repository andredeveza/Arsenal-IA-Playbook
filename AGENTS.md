# AGENTS.md

## Projeto
Arsenal IA Playbook

## Objetivo
Transformar o app em um produto vendável para usuários que nada sabem ou sabem pouco sobre IA.

## Público
Iniciantes em IA, profissionais não técnicos, pequenos empreendedores, autônomos e usuários que querem usar IA na prática.

## Direção do produto
O app não deve parecer uma biblioteca para experts.
Ele deve parecer um sistema guiado, prático e premium para aprender e usar IA no dia a dia.

## Prioridades
1. Clareza para iniciantes
2. UX simples e premium
3. Linguagem sem jargão
4. Não quebrar a base existente
5. Preservar visual premium
6. Melhorar onboarding, trilhas e usabilidade

## Stack
- React
- TypeScript
- Vite
- Zustand
- Wouter

## Arquivos importantes
- client/src/App.tsx
- client/src/data/mockData.ts
- client/src/store/userStore.ts

## Regras
- Não refatorar tudo sem necessidade
- Preferir alterações incrementais
- Preservar compatibilidade com a base atual
- Melhorar o produto com foco em iniciantes
- Evitar linguagem técnica e excesso de abstração
- Sempre explicar os arquivos alterados ao final

## Comandos
- instalar: npm install
- rodar: npm run dev
- build: npm run build
